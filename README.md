# Grofast Sales Dashboard

## Description
This Power BI dashboard visualizes sales data for the company ("Grofast Last Minute App"), providing insights into total revenue, average sales, item performance, and outlet establishment and location analysis.

## Dataset
The dashboard uses the following dataset:
* **Source:** CSV/Excel/SQL Database
* **Columns:** Date, Product, Region, Sales, Revenue, Item Fat Content, Item Type, Outlet Establishment Year, etc. (as suggested by the visual)

## Key Dashboard Insights
* **Total Sales:** $1.2 Million
* **Average Sales:** 141
* **Average Rating:** 3.92
* **Outlet Size Distribution:** Medium (1.75K), Small (3.63K), High (3.14K)
* **Top Outlet Locations (Tier 3):** 3.35K
* **Total Item List:** 9K

## Features
* Interactive charts and graphs for dynamic filtering.
* Regional sales comparison (Outlet Location).
* Product-wise revenue analysis (Item Fat Content, Item Type).
* Trend analysis over time (Outlet Establishment).

## Installation / Access
To view or edit the dashboard:
1.  Download the `.pbix` file from this repository.
2.  Open the file using **Power BI Desktop**.
3.  Refresh the data if required.

## Usage
* Click on charts (like Outlet Size or Outlet Location) to filter data dynamically.
* Use the filter panel (slicers) on the left to select specific Outlet Location, Outlet Establishment, or Item Type.
* Hover over visuals to see detailed information.

## Technologies Used
* **Power BI Desktop**
* **DAX** for calculated columns and measures
* **Power Query** for data transformation
